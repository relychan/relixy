# Relixy

Relixy is a monorepo of API gateway services that consists of:
- [RelyREST](https://relychan.com/docs/rely-rest/): an API gateway in front of your REST or GraphQL services with extra security and custom transformation.
- [RelyREST DDN](https://relychan.com/docs/rely-rest/hasura): a RESTified endpoint plugin for Hasura DDN.

## Documentation

Read the complete RelyREST documentation at [relychan.com/docs/rely-rest/](https://relychan.com/docs/rely-rest/)
